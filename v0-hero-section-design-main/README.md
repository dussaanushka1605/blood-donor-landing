## ğŸ“ Vercel v0 Usage Notes

The following components and sections were built using **Vercel v0**:

1. **Hero Section**: The complete hero section with responsive grid layout, heading, subtext, CTA buttons, and decorative image elements was generated using v0. The layout adapts beautifully from mobile to desktop with proper spacing and typography.

2. **Features Section**: The three-column features grid with icon cards, hover effects, and responsive design was created using v0. Each feature card includes an icon, title, and description with consistent styling.

3. **Testimonials Section**: The testimonials grid with profile images, feedback quotes, and role information was generated using v0. The section maintains visual consistency with the rest of the design.

4. **Contact Section**: The contact form with proper input fields, labels, and styling was built using v0. The form includes name, email, and message fields with focus states and proper accessibility.

5. **Overall Layout & Styling**: The responsive design patterns, color scheme (red theme for blood donation), spacing, and typography were all refined using v0 to ensure a cohesive and professional appearance across all sections.
